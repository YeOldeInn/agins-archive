SINGLE LEVEL QUEST CONTEST
RESTRICED TO ITEMS IN ORIGINAL GAME



The June contest will be to create a new single level quest. You are restricted to only the items found in the original game.

Entries should be emailed to me at dagin@my-deja.com or arkcaver@yahoo.com. If the mailbox gets full please drop me a note on the message board and I will fix the problem. If you cannot email a graphic quest to me and want to send a copy by postal mail, please email me first to make arrangements.

If you wish to be a judge in the contest, please contact me at one of the addresses above. A judge cannot submit an entry for this contest. You will be judging for this contest only. You can judge other contests, I just want all judges to sign up for each contest. This will allow someone to judge one contest and then enter another.

Good Luck!

CONTEST RULES
  1. Entries must be submitted by 8:00 a.m. Central Time (US) on November 1, 2001. Entries received after this time will not be considered.
  2. Entries using any item not included with the basic HeroQuest game will automatically be disqualified (although they can be submitted for the August contest). Items in the basic HQ game were: spell cards, monster figures, tiles, and furniture. Quests may have special rules, new monsters, new traps, etc., as long as they only use the items that came in the basic game. The 14 quests that came with the basic game are all good examples of this type of quest.
  3. If the quest calls for the Heroes to be a specific level (i.e. they should be experienced, they should have some artifacts, etc.), then this should be noted on the quest. Unless it is noted it will be assumed that these are general quests for all players.
  4. This quest should not be designed for use with the men-at-arms or mercenaries that came with some quest packs, as these were not included in the basic game.
  5. Entries may be submitted in any form, using any type of icon to represent the items in the quest. Scanned images of a paper quest is acceptable, as is a "text only" quest (text used to draw the map). I reserve the right to convert any scanned or text quest to the icons used on this website.
  6. All entries must be original, and not have been posted or published elsewhere previous to the contest.
  7. All entries will be posted to the website, on both this page and on the New Quests page.
JUDGING
Entries will be judged on the following criteria:
  • Originality - 45%
  • Enjoyment - 35%
  • Technical - 20%
Originality is just what it says - have you come up with a new idea for your quest. Is it special in some way? Is it different from other quests?

Enjoyment is the "fun factor". Was it fun to play? Was it too easy? Too hard?

Technical is graded on things like spelling, grammer, ease of understanding, whether the quest is consise, and so on.
