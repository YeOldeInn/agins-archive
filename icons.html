Below are files some files with icons
you can use to create your own quest maps.




Bitmapped HeroQuest Icons from the main game, plus several quest packs, are available from the
HeroQuest Resource site.

Non-graphic HeroQuest Icons (51k). These are the bitmap format (bmp) icons that I have used on many quests. I know it says they are non-graphical; it means that I used words instead of pictures to describe monsters, furniture, traps, and so on.

GIF format Icons (67k). These files contains GIF format tiles from the main game. They were created by Bob Terrell

Ray Bordier has made some Excell files for making quests.
  1. Questmaker Volume 1 (379KB)
  2. Questmaker Volume 2 (1,337KB)

DragonStrike Maps (22KB). Chaos Orc has created these bmp files for the Castle and the Meadow boards from DragonStrike.

DragonStrike Maps 2 (28KB). I created these bmp files for the Cave and the Meadow boards from DragonStrike.
